6. Story — VS Code chrome + webview mirror

The native VS Code status bar (bottom edge of the editor, rendered by src/status-bar.ts) and the webview footer mirror the same three states — idle, streaming, error — so the user always knows which mode is active regardless of which surface they last looked at. Mode changes trigger a 200ms flash on both sides, ensuring parity.
